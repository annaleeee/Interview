## Cascading (캐스케이딩)
** CSS (Cascading Style Sheet)

- Cascading (캐스케이딩)
  - 위에서 아래로 쏟아지는, 폭포 …  
  - 스타일 규칙 및 선언이 어떻게 적용되는지에 대한 규칙 및 우선 순위를 정의하는 프로세스
 
#### [ 원칙 ]
1. 스타일 우선순위
    - ‘중요도, 명시도, 코드 순서’를 통해 우선순위 결정
    - 중요도 (Importance)
        - 작성자 스타일 시트 > 사용자 스타일 시트 > 사용자 도구 스타일 시트
        - 중요도 최우선으로 끌어올리기
            - !important 키워드 사용
    - 명시도 (Specificity)
        - 인라인 > id > class > 태그
    - 코드 순서 (Source Order)
        - 가장 마지막에 등장한 속성을 최우선으로 적용
    
2. 스타일 상속
    - 부모 요소의 스타일을 자식 요소가 따라감
