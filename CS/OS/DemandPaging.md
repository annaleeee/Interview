## 요구 페이징
- 요구 페이징
    - 운영체제에서 사용되는 가상 메모리 관리 기법 중 하나
    - 프로세스를 메모리에 적재할 때, 처음부터 모든 페이지를 적재하지 않고 필요한 페이지만 메모리에 적재하는 기법
        - 실행에 요구되는 페이지만 적재
    - 필요하지 않은 페이지는 디스크에 저장하여 메모리를 절약하는 방법
      
- 장점
    - 효율적인 메모리 사용
    - 빠른 프로세스 실행
    - 대용량 메모리 지원
    - 프로세스 독립성
    - 유연한 메모리 할당
    - 높은 처리량
