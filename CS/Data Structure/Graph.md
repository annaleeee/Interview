## Graph
- 그래프 ( Graph )
  - 연결되어 있는 원소 간의 관계를 표현한 자료구조
  - 연결할 객체를 나타내는 정점( Vertext )과 객체를 연결하는 간선( Edge )의 집합으로 구성됨

#### [ 종류 ]
- 무방향 그래프
  - 간선에 방향이 없는 그래프 → 양방향 관계
- 방향 그래프
  - 간선에 방향이 있는 그래프 → 일방향 관계
- 가중치 그래프
  - 간선의 비용, 거리, 시간 등을 나타냄
- 이분 그래프
  - 무방향 그래프에서 노드를 두 그룹으로 나눔 <br>
    → 같은 그룹 내의 노드는 서로 인접하지 않고, 다른 그룹의 노드와만 인접하는 그래프
- 비순환 그래프
  - 사이클이 없는 그래프
