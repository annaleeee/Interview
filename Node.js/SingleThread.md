## Node.js는 싱글 스레드? 멀티 스레드?
- Node.js는 비동기 처리를 위한 `싱글 스레드` 사용
  - 멀티 스레딩이 가능은 함
      - 그렇지만, Node.js라는 환경을 이루는 특징이나 핵심 요소가 '이벤트 루프 + 메인 스레드'이며, 이러한 핵심 요소가 여러개 생기는 방식이 아니므로 개념적으로만 싱글 스레드라고 지칭
      - Worker_Thread로 멀티 스레딩 가능
          - 별도의 스레드를 생성하여 빠르게 병렬처리를 할 수 있는 기능
          - Node.js 환경에서 돌아가기는 하지만, 해당 스레드가 생성될 때 이벤트 루프가 하나 더 생기거나 이와 상호작용하는 스레드가 더 생기는 것은 아님
          
          ⇒ Node.js 자체는 여전히 싱글 스레드라고 함
            
    
    → '이론적으로는 멀티 스레드가 맞지만, 개념적으로는 싱글 스레드' 라는 주장
